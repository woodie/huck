# huck v0.3.1

## Right-click menu now has icons and a separator, matching zouk (#1)

`ScanThumbnailCell`'s right-click menu was plain text with no separator --
Compose Desktop's built-in `ContextMenuArea`/`ContextMenuItem` API has no
support for either. It's replaced with a manually triggered
`DropdownMenu`, matching zouk's real four-item `.contextMenu` exactly:
Download and Open, Download to…, and Fast Download each get an icon
(`Icons.Filled.OpenInNew`/`CloudDownload`/`FileDownload`), and a divider
now sits ahead of the destructive Move to Trash item at the bottom.

The toolbar's refresh button and the footer's delete button
(`CircularIconButton`) also shrank a touch, matching a direct ask to make
both a little smaller.

## `make check` is quiet again

`make check` used to reprint the exact same nested test tree `make test`
does -- no different from running `test` with an extra lint pass first.
It's now genuinely terse: silent (`PASS`) on a clean run, full log dumped
only on failure, matching the `test`/`check` split every other repo in
this account already uses. `make test` itself is unchanged, still the
full nested output.
